Important HTML tags

This page is designed as a quick reference for some of the common tags covered in Learn Enough HTML to Be Dangerous

The tables below don't include all HTML tags but they do list many of the important ones.

Block Elements

Tag Name Purpose
h1–h6 headings include a heading (levels 1–6)
p paragraph include a table
table table include a table
tr table row include a row of data
th table header make a table header
td table data include a table data cell

Inline Elements

Tag Name Purpose Example Result

strong strong make strong text <strong>at least a billion people</strong> strong
a anchor make hyperlink <a href="http://learnenough.com">Learn Enough</a> Learn Enough
img image include an image <img src="http://bit.ly/1MZAFuQ" alt="Michael Hartl">

Document Tags

Tag Name properties
html hyper text markup language represents the top(root) level of an HTML document
head head provides metadata about a document
body body represents the content of an HTML document
